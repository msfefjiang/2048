# 2048 Web Game

经典 2048 数字合并游戏的纯 HTML 实现，支持三种棋盘尺寸。

## 游戏版本

| 文件 | 棋盘尺寸 | 说明 |
|------|----------|------|
| [2048.html](2048.html) | 4×4 | 经典版本 |
| [2048_5x5.html](2048_5x5.html) | 5×5 | 加大版本 |
| [2048_6x6.html](2048_6x6.html) | 6×6 | 挑战版本 |

## 特性

- 🎨 支持亮色/暗色主题切换
- 📱 响应式设计，支持移动端触摸操作
- 🎮 键盘方向键 / 触摸滑动操控
- 💾 单文件 HTML，无需服务器，双击即玩

## 玩法

1. 直接用浏览器打开任一 HTML 文件
2. 使用方向键或滑动合并相同数字的方块
3. 目标：合成 2048！
